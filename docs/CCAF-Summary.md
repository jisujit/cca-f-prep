# The Claude Certified Architect — Foundations (CCA-F)

Launched by Anthropic on March 12, 2026, as part of its Partner Program, the Claude Certified Architect — Foundations (CCA-F) is the company's first official technical certification designed for professionals who build production-grade AI applications using Claude. It focuses on system design, agentic workflows, MCP tool integration, and reliability, rather than basic prompting. 

## Key Certification Details

- **Focus**: Technical architecture, agentic loops, Claude Code configuration, and RAG systems.
- **Structure**: 120-minute exam featuring 60 questions, requiring a 720/1000 score.
- **Target Audience**: Software engineers, AI engineers, and system architects.
- **Requirements**: Requires hands-on experience with the Claude API.
- **Availability**: Initially available primarily for Anthropic partners (Accenture, Cognizant, Deloitte, Infosys), with a $99 cost mentioned for general availability.

## Exam Topics (CCA-F)

1. **Agentic Architecture & Orchestration (27%)**: Designing multi-agent systems and Claude Code.
2. **Prompt Engineering & Structured Output (20%)**: Advanced prompting strategies and eval frameworks.
3. **Claude Code Configuration & Workflows (20%)**: CI/CD pipelines for AI.
4. **Tool Design & MCP Integration (18%)**: Building with Model Context Protocol (MCP).
5. **Context Management & Reliability (15%)**: Managing context and graceful failure.

## Preparation Tips

- **Hands-on Experience**: The exam requires deep knowledge of API authentication, system prompts, and tool calling.
- **Use Resources**: Review sample questions and the 60-question practice test.
- **Key Areas**: Focus on agentic workflows, MCP integration, and the Claude Code tool.
